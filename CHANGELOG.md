# Changelog - ExtractorOCR v1.0

## Mejoras Implementadas

### Actualización 1.0.1 - Progreso Detallado
- ✅ Agregado callback de progreso en OCRExtractor
- ✅ Progreso por página durante procesamiento
- ✅ Log detallado de cada paso:
  - "Dividiendo PDF en páginas..."
  - "Procesando página X/Y..."
  - "Página X completada"
  - "Procesamiento completado: Z páginas procesadas"
- ✅ Barra de progreso actualizada en tiempo real
- ✅ Mensajes informativos en cada etapa

### Características Actuales
- Configuración de 3 carpetas (Input, Processing, Output)
- División automática de PDF en páginas
- OCR con Gemini Vision API
- Generación de JSON 1 (Raw) y JSON 2 (Structured)
- Interfaz gráfica intuitiva
- Log en tiempo real
- Manejo de errores robusto

### Próximas Mejoras
- Traducción automática de textos
- Conexión a Base de Datos
- Exportación directa a SQL
- Filtros de búsqueda en resultados
- Vista previa de documentos
